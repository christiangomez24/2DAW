Se pide realizar un script con JavaScript que muestre por pantalla círculos de diferentes colores y diámetros en posiciones aleatorias de la pantalla (pero en el área visible). Para ello se debe emplear POO, creando una clase que tenga, al menos, las propiedades siguientes: Tamaño Color Posición (X,Y) y al menos un método constructor: para inicializar/calcular las propiedades necesarias pintar_circulo(), el cual pintará el círculo por pantalla El número de círculos que aparecerán debe estar parametrizada mediante una sola variable, la cual podrá ser introducida por el usuario (haciendo uso de prompt())
